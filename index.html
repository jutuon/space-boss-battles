Notes

  • Game requires WebAssembly and WebGL support.
  • Currently there is no music included with the game.
  • If game does not work or frame rate goes below 60 fps, try the game with latest Firefox and/or Google Chrome.
  • Currently only supported input devices are keyboard and mouse. Game controllers may work.
  • Controls are described at table below. Game's default pause key, Escape, is disabled in Emscripten build. Other keys listed in game's readme file also work, but some key combinations are web browser hotkeys.

Controls

Key Action
Arrow keys Move
Spacebar Shoot
P Pause game
Enter Select

How to enable full screen mode

  1. Select "Settings"
  2. Select "Full screen"
  3. If game resizes hit spacebar from your keyboard (or some other key that isn't Enter or Escape)
